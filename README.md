**Bonfire** is a Minecraft plugin that lets you easily claim chunks and manage how others can interact with them, including claim limits adapting to your playtime and BlueMap markers. For getting it and to find out more about it, check out [Modrinth](https://modrinth.com/plugin/bonfire)!


### Contributing
Feel free to open an issue if you have found a bug or want to suggest a feature! You're also welcome to contribute to the project, just open a pull request with your changes or additions. To have your changes merged, ensure they are complete, well documented and thoroughly tested!

Bear in mind that I am responsible for maintaining all new features and providing ongoing support, so it's always possible that I don't accept a pull request for any reason.
